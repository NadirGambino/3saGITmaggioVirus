Come rimuovere adeguatamente un Dialer

Ci sono alcune cose che devi fare nell'ordine in cui sono proposte:



  1. Trovare il file incriminato e cancellarlo dal computer.

    2. Per eliminare un file dal computer occorre prima conoscere il suo nome e la cartella in cui è presente. Se è presente una "ICONA" sulla scrivania del computer, cioè il desktop, il nome del file incriminato lo si trova in questo modo:

    1. col destro del mouse scegliere "proprietà";
    2. leggere le parole scritte nella casella di testo "Destinazione" e scriversele su un foglio, che si servono dopo; per esempio sarà: c:/..../xxxxxx.exe; dove xxxx è il nome del file ignoto, che occorre cancellare.
    3. Ora occorre trovare il file; basta scegliere dalla suddetta finestra "trova destinazione" e si viene portati proprio sul file incriminato; oppure dal menu "START" scegliere "trova" e poi trovare la cartella in cui è presente il file incriminato
    4. eliminare il file col destro del mouse, scegliendo elimina; in tal modo il file andrà nel cestino, che poi occorre svuotare, anche dopo.

    Un altro modo per conoscere il nome di un file in esecuzione è quello di premere contemporaneamente i tasti "Ctrl"+"Alt"+Canc" e leggere i nomi dei file in esecuzione. A questo punto il file dovrebbe essere stato eliminato dal computer ma esso è ancora attivo, e può riprodursi con un altro nome diverso, per cui occorre subito procedere al punto seguente.



  2. Eliminare il file dal registro di configurazione di Windows

    3. Il registro di windows contiene dei file che vengono eseguiti automaticamente. Questa procedura deve essere eseguita da una persona esperta, in quanto potrebbe bloccare il computer, se si compiono degli errori. Ma esperti si diventa, non si nasce, quindi proseguiamo con questo punto. Per accedere al registro di windows scegliere dalla scrivania di Windows:

    1. "START"
    2. "ESEGUI"
    3. scrivere "regedit" e scegliere OK
    4. Appare un editor per modificare il registro; l'editor ci mostra le varie cartelle che contengono i file del registro; il nome del file xxxxx.exe si trova nella cartella "RUN" che ora dobbiamo trovare seguendo il percorso indicato:

    HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Run

    Per cui occorre fare click col sinistro del mouse su ognuna delle cartelle fin quando non si arriva alla cartella "Run" che contiene il file incriminato; quindi click su:

    HKEY_LOCAL_MACHINE

    Software

    Microsoft

    Windows

    CurrentVersion

    Run

    Arrivati nella cartella "Run" occorre trovare la linea in cui ci dovrebbe essere il file xxxxx.exe; eliminare l'intera riga, col destro del mouse, scegliendo "elimina".



  3. Eliminare il file dall'esecuzione automatica

    4. Dalla scrivania di Windows scegliere:

    1. "START"
    2. "ESEGUI"
    3. Scrivere "msconfig" e premere "OK"
    4. Scegliere "ESECUZIONE AUTOMATICA"
    5. Deselezionare il segno di spunta Ö sulla linea in cui è presente il file xxxx.exe
    6. Scegliere "OK" e riavviare il computer


  4. Cancellare la icona del dialer dalla scrivania col destro del mouse


  5. Può darsi che il dialer si sia anche installato nelle connessioni remote, per cui occorre controllare tutti i numeri telefonici delle connessioni dai "BROWSER" e vedere se corrispondono a quelli indicati dal provider.


  6. Aggiornare ogni mese Windows collegandosi al sito:
    7. Microsoft Update
